Asking a question on Stack Overflow - simple version


The Title

Use "MS Access" rather than just Access to avoid confusion with access control posts. Show the language your code is written in for example VBA, Python, C#, SQL If your question does not involve code then perhaps you should post it on Super User

The Question

Post the code that is not working in the question so we can try it out. If you are getting an error message then write it out in full in the question body. Do not use a screenshot. Google the error message in full first before asking your question. Post the code that is not working in the question so we can try it out.

The Tags

Use the tags "MS Access" "VBA" or "SQL" and if the back end database is not MS Access then tag that for example - "SQL Server", "MySQL". If you are using VBA in AutoCAD then tag "AutoCAD" and "VBA".
