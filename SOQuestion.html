Asking a question on Stack Overflow - very simple version


The Title

Show the language your code is written in for example VBA, Python, C#, SQL. If your question does not involve code then perhaps you should post it on Super User If the question is specifically a Sharepoint question, perhaps better asked at Sharepoint SE

The Question

“What did you want to happen?”

“What actually happened?”

If you are getting an error message then write it out in full in the question body. Do not use a screenshot.

Google the error message in full first before asking your question.

Post the code that is not working in the question so we can try it out (and make it work!).

“Describe what you’ve tried so far (including searches, etc.)”
