Northwind 2010


Northwind 2010 is a sample database converted to Access 2010 from the original Access sample database shipped with the product over 10 years ago. This sample database illustrates how Access can manage small business customers, orders, inventory, purchasing, suppliers, shipping, and employees. The database can generate different reports, and is a great showcase for learning and customizing Access databases.

Northwind Traders is a fictional company invented by Microsoft trading in food and drinks.

You can experiment with the data stored in Northwind, and use the forms, reports, and other database objects as models for your own database.

Nowadays you do not need an Access licence for your PC to run an Access database. Microsoft has packaged the basic run time system and give it away free so that all you need is a Windows PC.
