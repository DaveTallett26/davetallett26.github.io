Welcome! All these pages were created with GeoBuilder 1.1
The first thing I noticed when using GeoBuilder was the appallingly slow response time. I live in the UK and the response may be better in the US but it is virtually unusable here in the UK. It is probably a combination of the fact the it is a Java application which is slow on my machine, a 32MB 166MHz Compaq Deskpro and the speed of the transatlantic links to GeoCities.

If I had not been using it to help out a Homesteader and for testing the new GeoBuilder 2.0 I would have stopped right here
